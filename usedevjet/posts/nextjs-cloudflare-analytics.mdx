1. Create your Web Analytics Site

On your [Cloudflare dashboard](https://dash.cloudflare.com/), select Web Analytics from the list on the right. Click Add a Site under the Quick Actions menu on the right. Enter the domain where your website is hosted and click `next` to continue. A tracking snippet will be displayed in a textbox. Click the textbox to copy the tracking snippet.

2. Add the JS Snippet to your website. Remember to update the token with tour cloudflare site token.

```tsx:_document.tsx {8,9,10,11,12}
import Document, { Html, Head, Main, NextScript } from 'next/document'

class MyDocument extends Document {
  render() {
    return (
      <Html>
        <Head>
          <script
            defer
            src="https://static.cloudflareinsights.com/beacon.min.js"
            data-cf-beacon='{"token": "your-cloudflare-site-token"}'
          ></script>
        </Head>
        <body>
          <Main />
          <NextScript />
        </body>
      </Html>
    )
  }
}

export default MyDocument
```
